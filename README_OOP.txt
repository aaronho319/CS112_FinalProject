Stackulator OOP Version 04/23/2018

Usage notes:
---------------------------------------------------------------------------
- This application implements a stack-based calculator using OOP through functions and Nodes. It uses the post-fix notation to implement the calculator.

How to use:

1) Open terminal or command prompt
2) Run javac StackulatorOOP.java
3) Run java StackulatorOOP
4) Type in your sample input (e.g. 3 5 1 - +)
5) Press enter
6) Enjoy the program
